# 遺傳

互動式遺傳學習網頁，主題聚焦在細胞、染色體、基因、等位基因、性狀，以及親代與子代的遺傳關係。

## 網頁

GitHub Pages 網址：<https://joyce401502-ai.github.io/genetics-page/>

## 使用方式

直接開啟 `index.html` 即可使用。若要重新產生網頁，可執行：

```bash
python generate_遺傳.py
```

## 互動內容

- 關係地圖：理解細胞、染色體、基因、等位基因的層級。
- 孟德爾配對實驗室：調整兩位親代的基因型，立即看到子代比例。
- 親子關係卡：把父母提供等位基因與子代可能性連起來。
- 快速檢查：用小測驗確認概念是否真的懂了。
